```
npm install

node ./bin/wallet

Usage: wallet [options]

Options:
  -c                              Generates a new key-pair and saves it into the wallet file.
  -g <address>                    Get balance of address.
  -l                              Lists all addresses from the wallet file.
  -s <amount> <from> <to> <mine>  Send AMOUNT of coins from FROM address to TO. Mine block, when <mine> is true.
  -h                              Output usage information.

```
